# Ubuntu Chinese Font Fix

Install Chinese fonts without sudo in TUM Eikon.

为你的TUM机房电脑安装中文语言包

## Install

```bash
git clone https://github.com/andycr0w/ubuntu-chinese-font-fix.git


cd ubuntu-chinese-font-fix


chmod +x install.sh


./install.sh
```

## Uninstall

```bash
chmod +x uninstall.sh


./uninstall.sh
```
